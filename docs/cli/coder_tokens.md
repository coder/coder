# coder tokens

Manage personal access tokens
## Usage
```console
coder tokens
```

## Examples
```console
  - Create a token for automation:                                              

      $ coder tokens create 

  - List your tokens:                                                           

      $ coder tokens ls 

  - Remove a token by ID:                                                       

      $ coder tokens rm WuoWs4ZsMX 
```
